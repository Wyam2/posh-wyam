    about_PoshWyam

SHORT DESCRIPTION
    PoshWyam is a PowerShell Module used to make working with Wyam easier.

LONG DESCRIPTION
    Wyam is a highly modular and extremely configurable static content generator
    and toolkit. PoshWyam provides PowerShell cmdlets used to make working with
    Wyam easier.
